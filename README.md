# <center> Проект: Анализ резюме из HeadHunter

## Описание проекта

Проект будет состоять из четырёх частей:

1. Базовый анализ структуры данных

2. Преобразование данных

3. Разведывательный анализ данных

4. Очистка данных

## Задачи проекта

Отработать основные этапы работы с данными на примере датасета о резюме

## Информация о данных

[База данных HeadHunter](https://drive.google.com/file/d/1HairH7ANXAY_bj0Gwmj3GuKy4OPKM4cI/view?usp=sharing)

Датасет представляет из себя набор данных в формате csv, состоящий из 44744 строк

До преобразования данных DataFrame содержит 12 признаков:

- Пол, возраст
- ЗП
- Ищет работу на должность:
- Город, переезд, командировки
- Занятость
- График
- Опыт работы
- Последнее/нынешнее место работы
- Последняя/нынешняя должность
- Образование и ВУЗ
- Обновление резюме
- Авто

Дополнительно используется [таблица с выгрузкой курсов валют](https://drive.google.com/file/d/1REkpNcAs0h9IN88vISlhCxSx4V4Jj-gH/view?usp=sharing)

## Этапы работы над проектом

***1. Базовый анализ структуры данных***

- Чтение данных с помощью библиотеки Pandas.
- Вывод нескольких первых (последних) строк таблицы, чтобы убедиться в том, что данные не повреждены. 
- Знакомство со структурой.
- Вывод основной информации о числе непустых значений в столбцах и их типах в таблице.
- Вывод основной статистической информации о столбцах.

***2. Преобразование данных***

- Создание с помощью функции-преобразования нового признака **"Образование"**, который имеет 4 категории
- Создание двух новых признака **"Пол"** и **"Возраст"**.
- Преобразуем признак **"Опыт"** - создание признака **"Опыт работы (месяц)"**
- Преобразуем признак **"Город, переезд, командировки"** - создание трех отдельных признаков
- Создаем признаки-мигалки для признаков **"Занятость"** и **"График"**
Преобразуем данные об ожидаемой заработной платы в общий вид - новый признак **"ЗП(руб)"** - заработная плата в рублях

***3. Исследование зависимостей в данных***

- Анализируем распределение признака **"Возраст"**
- Рассматриваем распределение признака **"Опыт работы (месяц)"**
- Анализируем распределение признака **"ЗП (руб)"**
- Анализируем зависимость медианной желаемой заработной платы (**"ЗП (руб)"**) от уровня образования (**"Образование"**)
-Анализируем распределение желаемой заработной платы (**"ЗП (руб)"**) в зависимости от города (**"Город"**)
- Анализируем зависимость медианной заработной платы (**"ЗП (руб)"**) от признаков **"Готовность к переезду"** и **"Готовность к командировкам"**
- Анализируем зависимость медианной желаемой заработной платы(**"ЗП(руб)"**) от возраста (**"Возраст"**) и образования (**"Образование"**)
- На основе диаграммы рассеяния, оцениваем зависимость опыта работы (**"Опыт работы (месяц)"**) от возраста (**"Возраст"**)

***4. Очистка данных***

- Находим полные дубликаты и удаляем их
- Выводим информацию о пропусках
- Обрабатываем пропуски
- Производим ручную очистку данных
- строим распределение признака "Возраст" в логорифмическом масштабе
- Находим выбросы с помощью метода z-отклонения и удаляем их из данных

## Результаты работы

Данные преобразованы к удобному виду, произведен анализ зависимостей и очистка данных

## Выводы:

- Возраст большинства соискателей сосредоточен в интервале 20-49 лет
(межквартильный диапозон 27-36)
- Желаемая заработная плата большинства сосискателей расположена в границах 17 000 - 180 000 руб.
- Для соискателей с высшем образованием наблюдается наибольший уровень желаемой заработной платой, наименьший для среднего и средне специаьного образования. Считаю, что признак уровня образования важен для прогнозирования заработной платы
- Признак "Город" не особо важен
- У соискателей готовых к переезду и командировкам уровень желаемой заработной платы выше
